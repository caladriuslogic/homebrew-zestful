# Homebrew Tap for Zestful

AI agent notifications for macOS.

## Install

```bash
brew tap caladriuslogic/zestful
brew install --cask zestful        # macOS app
brew install zestful-cli           # CLI tool
```

## Upgrade

```bash
brew upgrade --cask zestful
brew upgrade zestful-cli
```

## Uninstall

```bash
brew uninstall --cask zestful
brew uninstall zestful-cli
brew untap caladriuslogic/zestful
```

More info at [zestful.app](https://zestful.app)
